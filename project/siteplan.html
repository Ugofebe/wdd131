Studiimate - Website Plan

Site Name

Studiimate (AI-Powered Learning Assistant)

Why this name? "Studiimate" combines "Study" + "Mate," reflecting an AI companion that helps students learn efficiently. It’s memorable, relevant, and brandable.

Optional Domain: studiimate.app (available)

Site Purpose

The Studiimate website will:

  • Showcase the app’s AI-powered syllabus-aligned learning features.
  • Provide demo screenshots, testimonials, and use cases.
  • Offer a sign-up for beta testing and updates.
  • Highlight future roadmap items (study groups, past question bank).

Scenarios (Target Audience Questions)

  1. "How does Studiimate personalize learning for my courses?" → Website explains AI syllabus-matching and topic-based chat.
  2. "Can I access past exam questions related to my topics?" → Future feature highlighted with a "Coming Soon" teaser.

Color Schema

Primary (#4A6BFF) → Headings, buttons
Secondary (#FF6B6B) → CTAs, highlights
Dark (#2E3440) → Body text
Light (#F8F9FA) → Background
Accent (#FFD166) → Secondary accents

Typography

Fonts:

  • Boldonese (serif) → Headings (h1, h2, h3)
  • Roboto (sans-serif) → Body text, buttons, navigation

Wireframes

Mobile View

Key elements: Logo, hero section (app screenshot + CTA), features list, footer.

Desktop View

Key elements: Full-width hero, 3-column feature grid, demo video embed, sign-up form.
